Applet testing page

Species-changing wall.

Display is interactive:

  • click and drag with mouse to move around window and change size (scale) of the display.
  • Hold ctrl key while resizing to change the density of the system.
  • Press any number key on keyboard while cursor is display to change the number of image shells;
  • Press 's' key to show scale (relative to original size).

We have found the resizing feature to work only with the Netscape browser. Strange behavior with Explorer.
